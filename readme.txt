== Description ==

This plugin adds fullwidth menu on top of every page with links that you choose. It may be sites with similar content, subdomain sites or whatever you want.

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Create new menu in "Appearance" - > "Menus"
1. Add menu
1. Save
1. Associate menu with "Similar sites menu" in Theme Locations and save
1. Creatu custom link and add it into your menu. Don't forget to save.
1. For customization you must go into "Settings" -> "Similar sites menu"
